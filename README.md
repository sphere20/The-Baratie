# The-Baratie

This is an anime-themed online food ordering application using only Java.

### As a CUSTOMER, one can:
1. Sign up and login, change password  
2. Leave feedback and view reviews  
3. Browse menu  
4. Place order  

### As an ADMIN, one can:
1. Login, change password  
2. Update menu  
3. View orders  
4. View reviews
